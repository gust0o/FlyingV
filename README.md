# FlyingV

Mini sito statico, nero su bianco, per contare i giorni che mancano al rientro
in patria.

## Dati

I nomi e le date sono in `script.js`, dentro l'array `people`.

```js
{ name: "nome", arrivalDate: "2026-08-10" }
```

Per chi e' sempre in Calabria:

```js
{ name: "Giulius", infinite: true }
```

## Pubblicazione

Il sito non richiede build: GitHub Pages puo' servire direttamente i file dalla
root della branch `main`.
